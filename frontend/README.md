# Frontend (Next.js)

This workspace is the **Crooked** Next.js application: public site, API routes, and embedded Sanity Studio.

Setup, scripts, and architecture are documented in the **[repository root README](../README.md)** and **[docs/](../docs/)**.

Common commands (from repo root):

```bash
npm run dev:next   # this workspace only
npm run build      # production build of this workspace
```

Environment variables: see [`../frontend/.env.example`](.env.example) and [`../docs/environment-and-deployment.md`](../docs/environment-and-deployment.md).
